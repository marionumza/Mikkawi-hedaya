POS Order Sync

A POS Order Sync is a tool to synchronize order between multiple sessions(when more than one active sessions are available). It provides a special feature such that the seller can send orders for other active sessions and the other session can load/received that order and validate the order by making payment. It makes the linkage between multiple session such that any running session can send order for other running sessions. So it will increase the efficiency and decrease the load from the sessions.

Salient Features
  • Seller can send orders to other active POS Sessions.
  • Seller can load order(s) that are sent by other POS Sessions to his POS Session.
  • Seller can download the order receipt while sending order to other POS Session.
  • Only those orders are available to load that belongs to current running POS Session.
  • If any order is available to load for current running session then load button will be highlighted automatically
  • Seller can see status and history for all quotations/order that are sent by him.
  • It makes the linkage between the sessions, to sync order between multiple POS Sessions.
  • Order send by any POS Session will be saved in Odoo backend as quotation.

POS Session View

Sending Order To Other Active POS

After adding products to the cart,if seller want to send this order to any other active POS Session, then seller can click on send button to send this order as quotation.

On clicking send button, a popup window will be shown as shown below.Seller can select POS here and send quotation as well as print quotation details. On clicking 'Send and Print' button, quotation will be send and quotation receipt will be generated automatically

This message will be shown, if order is successfully sent to other session.

Loading Received Orders

To load quotations(orders) that are sent by other POS Session to current running POS Session, click on load button. To update quotation list at any time, click on refresh button as shown below. If any quotation(order) is available to load for current running session then load button will be highlighted automatically

Seller can Filter quotation on the basis of quote id,customer and load quotation by click on quotation.

Order Loaded

Quotation History

Click this button to see history and state of all those quotations that are sent by current running POS Session.

This history popup will be shown on click history button.Seller can see state of all those quotations that are sent by his POS Session.
